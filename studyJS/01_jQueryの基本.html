jQueryの基本

  • 最もポピュラーなJavaScriptライブラリ
  • DOM操作に優れている
  • かつてprototype.jsと勢力を二分していたが、MSが推したためjQueryの天下になった？
  • 今でも、MooToolsなどライバルはいる
  • ユーティリティとしての利用だけならUnderscore.jsなどもある（軽い）
  • 便利過ぎてJavaScriptの学習には弊害があるかもしれない
    　⇒ 中で何をしているか？を、想像できる程度にJavaScriptを知っておく必要があると思う・・・

$(function () {})

  • /* jQueryを使わずに書くと */ window.addEventListener("load", function () {})
  • 初回のページロード時の処理を書くときに使える

$("#hoge")

  • /* jQueryを使わずに書くと */ document.querySelectorAll("#hoge")
  • 要素の選択に使える
  • ※ただし戻り値はjQueryオブジェクト
    　⇒ ノードリストをラッピングしたもの？

$("#hoge").css("color", "blue")

  • /* jQueryを使わずに書くと */ document.querySelector("#hoge").style.color = "blue"
  • 本当は複数要素を取得できるので、querySelectorAllで取得してfor文でループする感じ
  • jQueryなら簡単！
  • $("#cssTest").css("color", "blue")

$("#hoge").addClass("comment")

  • /* jQueryを使わずに書くと */ document.querySelector("#hoge").className += " comment"
  • 本当は複数要素を取得できるので、querySelectorAllで取得してfor文で（ｒｙ
  • 複数のクラスが設定されている時に、クラスを追加するときに恩恵大
    　⇒ 自力でやるなら、既に同じクラスが設定されていないかチェックも必要になる
  • クラスを外す時は、removeClassメソッドがある

    • 　⇒ これも細かいチェックを自力でやると面倒
  • jQueryなら簡単！
  • $("#addClassTest").addClass("comment")

メソッドチェーン

  • jQueryのメソッドはjQueryオブジェクトを返す
    　⇒ 戻り値に対してさらに処理を実行できる
    　　⇒ その戻り値に対してさらに（ｒｙ
  • とにかくjQueryは簡単！
  • $("#chainTest").css("background-color", "blue").addClass("comment").attr("title", "attr()でタイトル属性付与")
